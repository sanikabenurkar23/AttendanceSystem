# Student Attendance Management System
A Python-based Student Attendance Management System developed using SQLite.

## Features
- Add Student
- View Students
- Mark Attendance
- View Attendance Records
- Search Student
- Update Student Name
- Delete Student
- Calculate Attendance Percentage

## Technologies Used
- Python
- SQLite
- VS Code

## How to Run
1. Open the project in VS Code.
2. Open Terminal.
3. Run: python attendance.py
